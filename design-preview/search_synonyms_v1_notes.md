# 검색 동의어·별칭 사전 v1 근거 요약 (search_synonyms.json)

- 작성: 채굴팀 워커 (총괄팀장 지시)
- 산출: `data/search_synonyms.json` (version syn_v1)
- 용도: 검색 매칭 앞단 쿼리 확장. 사용자 입력 변형/약칭/구어 → 원세컨드 정식 용어(canonical)로 확장해 recall 향상.
- 원칙: 자료 근거 통용어만. 억지 동의어·오해 소지·담보명 생성 금지. 모호 단독약칭 제외.
- 매칭 권장: 입력 정규화(소문자화 + 공백 제거 + 특수문자 제거) 후 alias→canonical 조회. 회사 별칭은 app.html `_normCo`와 동일 규칙이라 정합.

## 카테고리별 근거

### 담보 13종 (canonical=cover_data_v3 정식 담보명)

- 신규 5담보: 간병인사용일당(간병인일당/사용일당), 간병인지원일당(지원일당), 간병인보험(카테고리)(간병보험/간병인), 정기보험(카테고리)(정기/순수보장), 경영인정기보험(경영인/CEO보험/법인정기). 간병인일당은 "사용"과 "지원"을 alias 단계에서 분리(사용일당↔사용, 지원일당↔지원)해 혼선 방지.

### 질환 3종
- 암(암보험·갑상선암·전이암·재발암·제자리암), 뇌혈관질환(뇌/뇌졸중/뇌출혈/뇌경색/중풍), 허혈성심장질환(심장/심근경색/급성심근경색/협심증). 3대 진단비 정의(암·뇌혈관·허혈성심장)와 순환계 정의 근거. 질환어로 검색 시 질환 정식명으로 확장 → 하위 담보(3대·순환계) 전이 확장 가능.

### 고지코드 7종 (압축형 → 점형)
- 상품명·feature 실제 표기 근거. "The 가벼운 355"→3.5.5, "간편한(335)"→3.3.5, "든든한 3N5"/"3N5 더간편"→3.N.5, "3.10.5 간편고지"→3.10.5. 압축형(355/335/3N5)으로 쳐도 정식 코드로 확장.

### 상품군 7종
- insurer_products group·themes 근거. 종신/건강/연금/변액/어린이(태아)/운전자/여성. 정기·경영인은 담보 카테고리에 별도 존재해 중복 회피.

### 회사 15종 (_COMPANY_ALIASES 정합)
- app.html `_COMPANY_ALIASES`에서 데이터 21사에 해당하는 항목만 이관. 데이터 표기명을 canonical로(예: 농협생명=canonical, NH농협생명/NH생명/농협을 alias). KB손보/라이나손보 등 소식지 약칭 정합.
