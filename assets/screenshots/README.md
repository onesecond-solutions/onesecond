# 홈 카드 미리보기 스크린샷

홈 화면(pages/home.html) 카드 본진에 박힐 메뉴 미리보기 이미지 자리.

## 파일명 본진

| 파일 | 메뉴 | 비율 | 해상도 권장 |
|---|---|---|---|
| `menu-board.png` | 현장의 소리 (FIELD Q&A) | 16:9 | 1280×720 |
| `menu-quick.png` | 빠른 실행 (QUICK MENU) | 16:9 | 1280×720 |
| `menu-myspace.png` | MY SPACE | 16:9 | 1280×720 |

## 박는 본진

1. 본 메뉴 진입 (예: `https://onesecond.solutions/app.html` 박음 → 현장의 소리 박음)
2. 스크린샷 박음 (브라우저 윈도우 본진 또는 본진 일부)
3. 16:9 비율 박음 (잘림 자리 박지 X)
4. 본 자리 PNG 박음
5. 파일명 정합 박음 (위 표 본진)

## 자동 placeholder 처리

이미지 박지 X 박힌 자리 = `pages/home.html` 박힌 자리 JS = 자동 onerror 본진 = 카드 본진 = "미리보기 준비 중" 텍스트 박힘. 박힘 본진 = 자동 제거.
