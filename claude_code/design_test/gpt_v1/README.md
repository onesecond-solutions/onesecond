## 디자인 원칙

1. **흰 85% / 갈색 15%** — 색 비율
2. **활성 표시 한 곳만 강조** — 시선 유도
3. **페이지 정체성에 맞는 시각 언어 분기** — 무비판 복제 금지

## 발견된 디자인 자산 6종

1. **인포그래픽 컴포넌트 3종** — 인용 카드 / 숫자 비교 / 타임라인 (scripts, home)
2. **작업 도구 패턴** — 검색바 + 필터 + 정렬 (myspace, board)
3. **즐겨찾기 별표 패턴** — outline → filled 전환 (myspace)
4. **타임라인 점 + 핫 강조** — 좌측 컬러 점 + 활발한 항목 시각 강조 (board)
5. **이니셜 아바타** — 인물 사진 대체 (모든 페이지)
6. **C 사이드바 카드** — 라벨 + 제목 + 설명 3단 구조 (모든 페이지)
