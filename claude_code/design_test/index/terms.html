이용약관

원세컨드는 현재 무료 베타 운영 중입니다. 본 페이지는 베타 기간 임시 안내문이며,
정식 이용약관은 유료 전환 시점에 변호사 검수를 거쳐 게시될 예정입니다.

베타 기간 안내
  • 무료 회원 가입 시 모든 기능을 무상으로 이용하실 수 있습니다
  • 정식 이용약관 게시 시 기존 가입자에게 재동의를 요청드릴 예정입니다
  • 베타 기간 동안 서비스 내용·일부 기능이 변경되거나 일시 중단될 수 있습니다
  • 베타 운영상 발생하는 데이터·기능 오류는 신속히 수정하나, 그로 인한 영업 손실에 대해 책임지지 않습니다
  • 원세컨드는 단계별로 기능이 확장되는 베타 서비스입니다. 향후 CRM·이력 관리 등 신규 기능 도입 시, 해당 기능 사용 전 별도 옵트인 동의 절차를 거칩니다
  • 문의 사항은 카카오톡 오픈채팅으로 연락 주시기 바랍니다
