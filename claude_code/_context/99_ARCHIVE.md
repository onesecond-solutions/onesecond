### 핵심 정석 기준
- `public.users.id` = `auth.users.id` (1:1, UUID 동일)
- `handle_new_user` 트리거가 `auth.users` INSERT 시 `public.users` row 자동 생성
- 회원가입 폼은 Auth signUp + 추가 프로필은 **PATCH(UPDATE)**, INSERT 아님
- RLS는 `auth.uid() = id` 기준 명확히
- 외래키는 `auth.users(id) ON DELETE CASCADE`
