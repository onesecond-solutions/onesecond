## 교체 방향 (핵심 원칙)

1. **자동 흡수·확산 인상 제거** — "결국 원세컨드의 지식이 됩니다" 같은 플랫폼 전체 흡수 표현 삭제.
2. **공유 = 매번 내 선택** 임을 드러냄(단방향 자동 확산 인상 제거).
3. **격리는 지켜진다** — 공유 안 한 자료는 나만 봄. 범위는 팀/지점까지(플랫폼 전체 아님).
4. 톤: 압박("머무르지 마라") → 선택·안심.
