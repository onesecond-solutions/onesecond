## 레이아웃 (위→아래 세로 스택)
1. 상단 — 대시보드 3카드(가로): 오늘 일정 N / 공지 N / 답변 대기 Q&A N (실 COUNT, guide 제외). 클릭 연결. ※잠정(과투자 금지).
2. 하단 — 기능형 달력: 헤더 [오늘][<] ym [>] + 보기탭 [월][주][일][목록]. 월 그리드(시간+제목 한 줄, 유형색 최소). 날짜 클릭→우측 슬라이드 패널 즉시입력(제목·시간·메모·알람). 일정 클릭→같은 패널 수정/삭제. **팝업 남발 금지**.

## 알람 (사용자 선택 타이밍 · 인앱 → PWA 2단계)
3. 타이밍=사용자 선택: 정시/10·30·60분전/전날/없음. 복수 허용. 기본 없음.
4. 1단계 인앱: 시각 도달 시 종 센터 누적+팝업, 클릭 일정이동, 중복방지.
5. 2단계 PWA 푸시: 권한 동의+백그라운드 발송(SW/푸시서버). 착수 전 §4·인프라·비용 대표 결재. 분리 PR.
