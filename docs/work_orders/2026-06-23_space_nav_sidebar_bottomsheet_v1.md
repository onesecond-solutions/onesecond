## 1. 목표 (한 줄)
글로벌 사이드바에서 스코프를 고르면, **바텀시트(책상) 안에서** 그 스페이스 "페이지"(상단 배너 + 메뉴 카드 + 카드클릭 시 인플레이스 전환)가 렌더된다. 풀페이지 전환 없음.

```
사이드바(글로벌)        바텀시트(책상) 내부 — 안 닫힘
  보험사자료실(top·admin)   ┌ 배너 이미지(스코프별) ─────────┐
  ───────                   │ [카드][카드][카드]             │
  홈                        │  마이: 다이어리·자료실·스크립트·메모 │
  마이스페이스  ───클릭───▶  │  팀/지점: 공지게시판·스크립트·자료실 │
  팀스페이스                │ ├───────────────────────────┤ │
  지점스페이스              │ │ 카드 클릭 → 그 자리서 내용 전환 │ │
  스크립트                  │ │ (_spSplitShow 인플레이스 재사용)│ │
  매니저방                  └ └───────────────────────────┘ ┘
  ───────
  어드민(bottom·admin)
```

## 2. 사이드바 최종 구조 (위→아래)
| 항목 | 노출 | 비고 |
|---|---|---|
| 보험사자료실 | **admin만** | 최상단 유지(현행 게이트 그대로) |
| 홈 | 전체 | |
| 마이스페이스 | 전체 | 클릭=바텀시트 마이 페이지 |
| 팀스페이스 | 전체 | 클릭=바텀시트 팀 페이지 |
| 지점스페이스 | 전체 | 클릭=바텀시트 지점 페이지 |
| 스크립트 | 전체 | 글로벌 스크립트 뱅크(현행) |
| 매니저방 | 매니저/지점장/admin(is-mgr) | 현행 게이트 유지 |
| 어드민 | **admin만** | 최하단 유지 |

- **숨김(사이드바에서 제거):** 함께해요 · 보험이슈 · 요금제
- **흡수:** 다이어리 → 마이스페이스 카드로 (사이드바 단독 항목 제거)
- **간격 compact:** 항목 수 늘어난 만큼 줄간격/패딩 축소(토큰 기반)

## 3. 바텀시트 스페이스 페이지 구조
- 상단 **배너 이미지**(스코프별 1장: 마이/팀/지점)
- 하단 **메뉴 카드 그리드**
- 카드 클릭 → **기존 인플레이스 엔진 `_spSplitShow`(app.html:14523~) 재사용** → `#spContent`에 해당 콘텐츠 렌더. 페이지 이동 0.
- 기존 트리(#spTree)는 사이드바로 역할 이관(스코프 선택) → 시트 안은 배너+카드 중심.

## 4. 스코프별 카드 + 공유 라우팅
| 스코프 | 카드 | 비고 |
|---|---|---|
| 마이 | 다이어리 · 개인 자료실 · 스크립트 · 메모 | 메모=마이 전용 |
| 팀 | 공지게시판(공지+공유알림) · 스크립트 · 자료실 | |
| 지점 | 공지게시판(공지+공유알림) · 스크립트 · 자료실 | |

**공유 라우팅(1:1):** 스크립트 공유→스코프 스크립트 / 파일 공유→스코프 자료실 / **메모=팀공유 제외(개인전용)**. 공지게시판엔 풀본문 아닌 **"X가 공유함 +[보기]" 알림 포인터**만(오늘 막은 team_notices 풀콘텐츠 도배 해소 정합).
**공유함 노드 폐지** — 공유물은 해당 스코프 메뉴로, 알림은 공지게시판으로.
