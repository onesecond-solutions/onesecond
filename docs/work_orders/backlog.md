## 검수 큐 UX v2 — Phase 3 가동 직전 (1개 PR로 묶음, 2026-06-06 결재)

**배경:** 지식창고 Phase 1(PR #432) = 기본 검수 큐(행별 승인·보류·폐기 + 일괄 승인 + status 필터탭). UX 고도화는 야간 자동 학습(Phase 3) 가동 직전에 **1개 PR로 묶어** 진행. Phase 1 = 그릇·동작 검증까지, v2 = 대량 검수 효율화.

**항목 4 (`pages/knowledge-vault.html` 검수 큐 탭):**
1. **마스터-디테일 전환** — 좌 카드 목록 / 우 본문 전문 + 판정 버튼. 기존 게시판 카드 스타일 재사용. 모바일 = 목록→상세 전환.
2. **판정 후 자동 다음 항목 이동** — 승인/보류/폐기 직후 다음 ai_draft로 포커스 자동 이동(연속 검수 속도).
3. **전체 선택 / 현재 필터 전건 승인 버튼** — 기결재 C안. Phase 1엔 전체 선택 없음(개별 체크만) → 대량 승인 시 SQL 의존 해소.
4. **학습일지 → 검수 큐 바로가기** — 학습일지 탭에서 해당 run이 만든 ai_draft 항목으로 바로 점프(run 단위 검수 동선).
5. **Phase 3 정식 편입 = SPA 내부 통합 (종착지, 2026-06-06 기록)** — 현재 운영센터 지식엔진 탭은 외부 페이지로 링크(`target=_blank` 이동). **종착지는 링크 이동이 아니라 지식창고·검색을 지식엔진 탭 '안에서' 직접 렌더하는 SPA 내부 통합**(`#v-admin` 라우터에 view로 흡수, app.html 페이지 분리 해소). Phase 3 정식 편입 시점에 착수. **지금 착수 금지.**

## 검색 — 전체 통합 작업 (2026-06-11 예약, 착수 금지)

**배경:** 공동 스페이스 보험사별 자료실을 스마트 게시판 레이아웃으로 이식(PR #557)하면서 **검색(`toggleLocalSearch`)은 제외**함. 검색은 v-voice(현장의 소리)에 강하게 결합돼 있어, 개별 화면마다 붙이지 말고 **한 번에 전체 작업으로** 다룬다.

**예약 범위 (전체 검색 일감):**
- 공동 스페이스 보험사별 자료실 방 내부 검색
- 그 외 게시판형 화면들의 로컬 검색 일관화 (현장의 소리·자료실·MY SPACE 등)
- `toggleLocalSearch`의 v-voice 강결합 분리 → 컨테이너 독립형으로 재설계(공용화)

**전제:** 개별 PR로 쪼개지 말고, 검색 UX를 전 화면 공통으로 한 번에 설계·이식하는 전체 작업으로 진입. 착수 시점·우선순위는 추후 대표님 결재.

## 보안 — 기존 지점/팀 합류 사칭 방지 (2026-06-12 분리 결재)

**배경:** 회원가입 위저드(GA 트리) 보안 검수(Gemini·Claude Chrome·Code + 실제 RLS 실측) 결과 도출. 자율 가입에서 **신규 조직 생성자는 RLS상 빈 조직으로 격리되어 안전**하나, **기존 지점/팀을 골라 합류하는 가입자는 그 조직의 branch_id를 받아 해당 게시판(네비방·Q&A·매니저공지 등)을 즉시 열람**할 수 있음. 이메일 OTP는 "이메일 소유"만 증명, "실제 그 지점 소속"은 증명 못 함 → **사칭 합류 시 그 지점 실데이터 노출.** 현재 시스템도 이미 그러함(즉시 active).

**범위 밖 결정:** 회원가입 위저드 1차 범위에는 **넣지 않음**(2026-06-12 A안 결재). 위저드는 신규 조직 생성·미배정 해소가 목적이고, 사칭 방지는 별도 보안 설계가 필요.

**후속 설계 후보:**
- 기존 지점/팀 **합류 시 = 그 지점/팀 매니저(또는 admin) 승인**(insurer엔 이미 `approve_insurer_user` 흐름 존재 → GA에도 적용 검토)
- 합류 직후 = 제한 열람(시드·공개만) → 매니저 승인 후 branch_id 부여(=실데이터 열람)
- 또는 초대 코드/링크 기반 합류(매니저가 발급) = 사칭 원천 차단

**전제:** 자율 입점(수요 레이어)의 마찰 trade-off를 고려해 착수 시점·강도는 추후 대표님 결재. 위저드 GA 트리 마감 후 진입 권장.
