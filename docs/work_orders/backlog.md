> 결재는 났으나 설계/우선순위로 뒤로 미룬 항목. 착수 시 이 파일에서 꺼내 진행하고, 완료되면 줄을 지운다.

## 어드민(운영센터) — 엔티티 카드 상세뷰 (2026-05-31 보류)

**배경:** 운영센터 SPA 흡수(PR #207~211) 완료. 대시보드 위젯·버튼 드릴다운까지 라이브. 단, **목록 카드 본체 클릭 = 비활성**(원본 admin부터 없음). "어드민은 이 정도에서 멈춤" 결재(2026-05-31) → 다른 페이지 마무리 후 재개.

**남은 작업 = 엔티티 카드 클릭 시 상세뷰** (설계 결정 선행 필요):
- **게시글 카드** → 현장의 소리 원문 딥링크 vs 어드민 내 상세 모달 (어디로?)
- **사용자 카드** → 상세 관리 모달 (권한 변경·정지·지점 재배정)
- **지점 카드** → 소속 인원 목록 상세
- **댓글 카드** → 원 게시글 딥링크
- **자료실 카드** → 자료 상세/원문

**참고:** 카드 클릭=내부 원본 딥링크 원칙([[home_no_mock_realdata_principle]] 정합). 재개 시 위 5종 목적지 결재부터.

**현 상태(라이브):** 메인테넌스 ON(대개편 검수 중)이라 외부 노출 없음 → 급하지 않음. 대시보드/탭·칩/스크롤/배지/데이터(실DB)는 정상.
