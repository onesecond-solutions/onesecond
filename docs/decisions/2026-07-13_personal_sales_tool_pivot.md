# 2026-07-13 원세컨드 방향 전환 — 개인 영업 도구 중심 재편

> **대표 확정(2026-07-13).** 번아웃·복잡성 피로 속 방향 재정의: **원세컨드 = 임태성 대표 본인 영업에 실제로 쓰는 도구.** 팀·지점·조직 계층은 전면에서 내린다. (이 문서 = 재편 트랙의 진실원천. 다음 세션이 조직 중심으로 회귀하지 않게 하는 회귀 방지 기록.)

## 배경
- 상담관리/고객관리 화면 정의를 두고 반복적으로 꼬임(조직·옛 대량 데이터·상태값 혼재). 대표 피로 누적.
- 대표 원문: "손 털고, 내 영업에 활용할 수 있는 사이트로 만들고 싶다. 사람들한테 질린다. 팀·지점 다 필요 없고 **범용으로 다 사용할 수 있는 사이트로 구조 개편**하고 싶다."

## 대표 확정 2결정
1. **조직 기능 = 숨기고 보존** — 팀·지점·조직 기능을 화면에서 내리되 **코드·데이터는 그대로 보존**. 나중에 필요하면 복구 가능, 위험 0. (완전 제거 아님.)
2. **범위 = 일단 나부터** — 대표 개인 영업 완성도 우선. 구조는 **조직 없는 '개인 단위'**(= 범용의 기반)로.

## 의미
- **새로 짓는 게 아니다.** 이미 핵심 기능 대부분(상담관리·고객관리·검색·소식지·로드맵·영업노트 등)이 임태성 게이트 전용 = 사실상 대표 전용. 조직 구조는 뒤에서 잠자는 뼈대. → **앞으로 끌어내고 조직 복잡함을 뒤로 접는 일.**
- **정체성과 정합.** '중립 독립 범용 SaaS'(0순위 정체성)에서 조직 계층을 걷어내고 개인이 바로 쓰는 범용 도구로 가는 건 오히려 그쪽에 더 가깝다.
- **파괴 없음.** 조직 코드·RLS·데이터 보존, 화면/네비에서만 숨김. 되돌리기 쉬움.

## 앞으로 (초안 — 각 단계 대표 실화면 확인 하에 진행)
1. **진행 중 개인영업 작업 마무리** — 상담관리/고객관리 흐름 정리 + 데이터 현황 디테일화(월별·유입별, `feat/consult-data-stats-detail` 진행 중).
2. **홈·사이드바 재편** — 대표 개인 영업 도구 중심으로 전면 배치, 조직 메뉴는 뒤로 숨김.
3. **조직 기능 숨김 정리** — 매니저방·팀/지점 공유·원수사 admin 등을 숨김 플래그로 정리(코드 보존).
4. **(나중) 개인 단위 범용 온보딩** — 대표 외 개방 시점·방식 결정.
