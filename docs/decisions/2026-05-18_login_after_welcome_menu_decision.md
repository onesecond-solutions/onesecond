## C 풀스크린 카피 확정안 (가입 직후 1회 노출)

```
━━━━━━━━━━━━━━━━━━━━━━━━━━
한 화면에 모두 배치

   [이름] 팀장님,
   반갑습니다.

   ─────────────────

   지금 보시는 원세컨드는
   제가 보여드리고 싶은 모습의
   20%입니다.

   ████░░░░░░░░ 20%

   ─────────────────

   [캡처 미리보기 공간]
   ← 사용자가 다음에 만날 화면 미리보기
   (이미지는 본인/Chrome AI가 채움)

   ─────────────────

   나머지 80%는
   여러분과 함께
   만들어 갑니다.

   ┌────────────────┐
   │  함께 시작하기   │  ← home_v2로 이동
   └────────────────┘
━━━━━━━━━━━━━━━━━━━━━━━━━━
```

**디자인 포인트:**
- 배경 = 원세컨드 브랜드 컬러 (아이보리/웜 톤)
- 글자 = 큼지막, 한 화면에 다 배치
- 진행 바 = 브랜드 컬러로 채워지는 애니메이션 (0% → 20% 차오름)
- **캡처 미리보기 공간** = 16:9 비율, 사용자가 다음 행동 인지하도록 시각 안내. 이미지 자체는 별도 채움
- "함께 시작하기" = 한 가지 CTA
- 가입 직후 첫 로그인 **1회만** 노출 (Supabase `users.onboarding_seen` flag)

```
┌──────────────────────────┐
│  헤더 (로고 + 메뉴)         │
├──────────────────────────┤
│                          │
│ 안녕하세요 [이름] 팀장님    │
│                          │
│ ─────────────────────    │
│ 원세컨드 완성도            │
│                          │
│ ████░░░░░░░░░ 20%       │
│                          │
│ 여러분이 쓰실수록 함께 완성  │
│ ─────────────────────    │
│                          │
│ (이하 home 콘텐츠)         │
└──────────────────────────┘
```

**% 변경 방식:**
- 팀장님이 채팅으로 "이제 22%로 올려줘" 요청 → Claude Code가 home.html 수정 + 배포
- 변경 빈도 = 월 1~2회 예상
- 향후 자동 계산 전환 가능 (실데이터 기반)
