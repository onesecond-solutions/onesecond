### 하(오류 확정) 목록

1. **불완전판매** — 원인(판매 부실 행위)과 결과 지표(해지·무효 건수)를 혼용
2. **적정성원칙** — 금소법 제18조 적정성원칙을 "보험료 납입 가능 수준" 원칙으로 잘못 기술 (법령상 적정성원칙 = 소비자 자발적 구매 시 부적정 고지 의무)
3. **품질보증해지** — "보험사의 품질보증 불이행"이라는 주체·원인 기술 오류 (실제 = 모집인 불완전판매를 보험사가 품질보증 명목으로 해지)
4. **법인보험대리점** — "보험중개 기관"이라는 표현 오류 (보험업법상 대리점 = '대리', 보험중개사 = '중개'; 혼용 금지)
5. **승환계약** — 적립금 충당 = 계약전환(Contract Conversion)에 해당하는 설명. 승환계약(Replacement)은 기존계약 해지 + 신계약 체결이며 적립금 충당 항목은 일반 정의에 불포함

## 주요 수정 권장 (우선순위)

### 우선순위 1 — 오류 확정 (즉시 수정)

1. **적정성원칙** ★★★  
   - 현행: "보험료 수준이 소득 수준에 비해 과도하지 않고 지속 납입 가능한 수준이어야 한다는 원칙"  
   - 문제: 금소법 제18조 적정성원칙과 전혀 다른 개념을 기술  
   - 수정: "소비자가 먼저 구매 의사를 밝힌 경우, 보험모집인이 소비자 정보를 파악하여 해당 상품이 부적정하다고 판단되면 그 사실을 고지해야 하는 금융소비자보호법(제18조)상 판매 규제 원칙."

2. **승환계약** ★★★  
   - 현행: 적립금 충당 = 계약전환 설명 혼입  
   - 수정: "기존 보험계약을 해지하고 새로운 보험계약을 체결하는 행위. 모집인이 부당하게 기존 계약을 소멸시키고 신계약 체결을 권유하는 부당승환은 보험업법에서 금지됨."

3. **법인보험대리점** ★★  
   - 현행: "보험중개 기관"  
   - 수정: "보험중개 기관" → "보험계약 체결을 대리하는 법인 기관" (중개 ≠ 대리 혼용 제거)

4. **품질보증해지** ★★  
   - 현행: "보험사의 품질보증 불이행"  
   - 수정: "보험모집인의 불완전판매가 확인된 경우 보험회사가 소비자 보호 차원에서 계약을 해지하는 제도"

5. **불완전판매** ★★  
   - 현행: 결과(해지·무효) = 불완전판매 자체로 기술  
   - 수정: 원인(판매 과정 부실)과 결과(측정 지표) 분리 기술
