## §4. 중복 의심 통합 목록 (적재 단계 dedupe 대상)

배치 간 동일/유사 용어가 다중 출현 — 적재 시 1건으로 통합 + 대표 카테고리 지정 필요:

- **피보험자 / 계약자 / 수익자(보험수익자)** — 계약·보장·판매·민원 4중 출현
- **약관(보험약관)** — 계약·보장·판매·민원
- **보험료 / 보험금** — 계약·보장·판매
- **보험기간 / 보험증권** — 계약·보장·판매
- **해지환급금(해약환급금) / 만기환급금 / 납입면제 / 배당·무배당** — 보장·계약·제도
- **고지의무 / 보험사기 / 면책사유 / 보험계약 무효** — 계약·판매·민원·제도 산재
- **특약 / 갱신형보험 / 정기보험 / 종신보험** — 계약·보장·제도
- **역선택** — 인수·민원 (양쪽 다 '하·오류')
- **상품설명서 ↔ 핵심상품설명서**, **설명의무 ↔ 대면설명**, **완전판매모니터링 ↔ (해피콜)** — 동일/상하위 관계 통합 검토

→ 중복 제거 시 실제 고유 용어는 293건보다 상당히 적을 것(추정 230~250건). 정확 dedupe는 적재 단계.

## §5. 건수 차이 사유

### 채굴 293 / 목표 300 (−7)
워커가 "정확히 50개"를 못 맞춤 — 미달(01=45·02=43·04=45=−12)과 초과(03=53·06=57=+10, 05=50)가 상쇄되어 순 −7. 부족분 7건 보충은 별도 지시(이번 사이클은 검수만).

### 검수 276 / 채굴 293 (−17, 검수 커버리지 갭)
검수 워커가 초안 일부 용어를 누락(03=−8, 04=−1, 05=−3, 06=−5). 원인 추정: 초안 내 중복·동일명 용어를 1건으로 병합 처리했거나 표 작성 중 누락. **미검수 17건은 다음 라운드 보완 또는 dedupe 단계에서 흡수 대상.**
