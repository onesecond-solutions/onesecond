- 설계사 현장 빈출 **기초 용어** (계약·인수·보장·실손·세제 등 일반 보험 용어)
- **중립 독립 SaaS 정합** — 특정 보험사 상품명·고유 명칭 배제, 일반 용어만

- 출처 없는 항목은 **만들지 않음** — 출처 미확보 시 보류 처리

## 적재 시 매핑 메모 (향후 별도 지시)
- `knowledge_entries`: `type='term'`, `status='ai_draft'`(검수 큐 경유), `source_type='official_glossary'`,
  `source_title`=출처명, `created_by='admin'`.
- 동의어 후보 → `knowledge_synonyms`(쌍별 결재 = 게이트), 본 초안에서는 후보만 표기.
- **라이브 최종 중복 대조는 적재 단계에서 팀장님 SELECT로 재확인** (본 초안 기준선은 마이그레이션 파일).
