### 청약
- 정의: 보험계약자가 보험회사에 대하여 일정한 보험계약을 체결할 것을 목적으로 하는 의사표시로서, 이를 보험회사가 승낙하면 계약이 성립한다.
- 분류: 계약
- 키워드: 계약성립, 의사표시, 의도, 보험회사
- 출처: https://easylaw.go.kr/CSP/CnpClsMain.laf (찾기쉬운 생활법령정보 — 검수 시 경로 확인)

### 승낙
- 정의: 보험회사가 보험계약자의 청약에 대해 계약 체결에 동의하는 의사표시로, 청약과 승낙이 만나면 보험계약이 성립한다.
- 분류: 계약
- 키워드: 의사표시, 동의, 계약체결, 보험회사
- 출처: https://easylaw.go.kr/CSP/CnpClsMain.laf (찾기쉬운 생활법령정보 — 검수 시 경로 확인)

### 고지의무
- 정의: 보험계약자 또는 피보험자가 보험계약 체결 시 고의 또는 중대한 과실로 중요한 사항을 알리지 않거나 부실의 고지를 하지 않을 의무를 말한다.
- 분류: 계약
- 키워드: 의무, 고의, 과실, 중요사항
- 출처: 출처 미확인 (블로그 출처 — 검수 시 공식 출처 대조 필요)

### 계약전알릴의무
- 정의: 보험가입자가 보험회사에 계약 체결에 중요한 영향을 미치는 사항(고지사항)을 반드시 알려야 하는 법적 의무이다.
- 분류: 계약
- 키워드: 공시, 의무, 건강상태, 직업, 위험
- 출처: 출처 미확인 (보험사 PDF — 검수 시 공식 출처 대조 필요)

### 청약철회
- 정의: 보험계약자가 보험증권을 받은 날부터 15일 이내, 청약한 날부터 30일 이내에 특별한 사유 없이 청약을 철회하여 청약 이전의 상태로 되돌리는 것이다.
- 분류: 계약
- 키워드: 철회, 환급, 기간, 권리, 소비자보호
- 출처: https://www.easylaw.go.kr/CSP/CnpClsMain.laf (검수 시 경로·기간 수치 확인)

### 부활
- 정의: 보험료 연체로 실효된 보험계약을 다시 살리는 제도로, 실효된 달에 밀린 보험료와 이자를 납부하면 계약을 복구하는 것이다.
- 분류: 계약
- 키워드: 실효회복, 연체료, 이자, 계약복구
- 출처: 출처 미확인 (블로그 출처 — 검수 시 공식 출처 대조 필요)

### 실효
- 정의: 보험료가 연속 연체될 경우 일정 시점부터 보험계약은 유지되지만 보험의 효력이 상실되는 상태이다.
- 분류: 계약
- 키워드: 연체, 효력상실, 보장중단, 기간
- 출처: 출처 미확인 (블로그 출처 — 검수 시 공식 출처 대조 / 연체 개월 수치 확인 필요)

- 정의: 보험료 납입이 어려울 경우 회사가 해약환급금 범위 내에서 보험료를 자동으로 대출금으로 처리하여 납입하는 제도이다.
- 분류: 계약
- 키워드: 자동이월, 대출, 해약환급금, 유지

- 분류: 계약

- 분류: 계약
- 키워드: 대상자, 보장대상, 계약자, 수익자

### 계약자
- 정의: 보험계약을 체결한 사람으로 보험료 납부 의무를 가지며, 계약 전 건강 상태 등 중요한 사항을 미리 고지할 의무가 있다.
- 분류: 계약

- 분류: 계약
- 키워드: 수령자, 보험금, 지정, 변경
- 출처: 출처 미확인 (보험사 페이지 — 검수 시 공식 출처 대조 필요)
