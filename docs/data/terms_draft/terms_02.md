# terms_02 — 보장/담보 (채굴 직원 2, 1차 사이클)

> 검수 전 초안(ai_draft). 출처·정의 미검증. 검수자(팀장 코드)가 외부 출처 대조 후 confidence 부여.

### 담보
- 정의: 보험 약관에서 보험회사가 보장하기로 정한 보험사고 항목으로, 보험금을 지급하는 기본 단위입니다.
- 분류: 보장
- 키워드: 보험사고, 보험금 지급, 약관, 계약 범위
- 출처: https://fine.fss.or.kr/main/prc/is/sub/is004.jsp

### 보장
- 정의: 보험계약에서 보험회사가 보험료를 받고 약정한 보험사고에 대해 보험금을 지불할 책임을 말합니다.
- 분류: 보장
- 키워드: 책임, 보험금, 약정, 위험 전가
- 출처: https://fine.fss.or.kr/main/prc/is/sub/is004.jsp

### 진단비
- 정의: 암 또는 질병으로 진단받았을 때 지급되는 보험금으로, 치료비 부담을 완화하기 위한 보장입니다.
- 분류: 보장
- 키워드: 질병 진단, 암, 보험금, 의료비 보조
- 출처: https://fine.fss.or.kr/main/prc/is/sub/is004.jsp

### 입원일당
- 정의: 피보험자가 의료기관에 입원한 기간에 대해 하루 단위로 지급되는 보험금입니다.
- 분류: 보장
- 키워드: 입원, 일당, 일수 기준, 병원
- 출처: https://fine.fss.or.kr/main/prc/is/sub/is004.jsp

### 수술비
- 정의: 의료기관에서 수술을 받을 때 발생하는 수술료 및 관련 비용을 보장하는 보험금입니다.
- 분류: 보장
- 키워드: 수술, 의료비, 수술료, 관련 비용
- 출처: https://fine.fss.or.kr/main/prc/is/sub/is004.jsp

### 실손의료비
- 정의: 피보험자가 질병이나 상해로 인해 의료기관에서 실제로 부담한 의료비를 보상하는 보장입니다.
- 분류: 보장
- 키워드: 실제 부담, 의료비, 보상, 입원·통원
- 출처: https://kpub.knia.or.kr/productDisc/lostHealth/lostHealthInfo.do

### 통원비
- 정의: 의료기관에 입원하지 않고 방문하여 진료를 받을 때 발생하는 의료비를 보상하는 보장입니다.
- 분류: 보장
- 키워드: 외래진료, 방문 진료, 외과진료, 의료비
- 출처: https://kpub.knia.or.kr/productDisc/lostHealth/lostHealthInfo.do

### 후유장해
- 정의: 질병이나 상해로 인한 치료가 종료된 후에도 신체 기능이 완전히 회복되지 않아 영구적 손상이나 장애가 남은 상태를 말합니다.
- 분류: 보장
- 키워드: 영구 장애, 신체 손상, 후유증, 장해율
- 출처: https://www.hanalife.co.kr/csc/accidentGuideRenew/accidentReason_2.do

### 사망보험금
- 정의: 보험기간 중 피보험자가 사망하거나 사망한 것으로 정부기관이 인정한 경우에 지급되는 보험금입니다.
- 분류: 보장
- 키워드: 사망, 사망 보장, 상속, 수익자 지급
- 출처: https://www.hanalife.co.kr/csc/accidentInsuranceGuide/accidentInsuranceRequiredDocumentGuideDeath.do

### 만기환급금
- 정의: 보험기간이 만료되었을 때 납입한 보험료의 일부 또는 전부를 계약자에게 돌려주는 금액입니다.
- 분류: 보장
- 키워드: 만기, 환급, 보험료 반환, 계약 종료
- 출처: 출처 미확인 (블로그 출처 — 검수 시 공식 출처 대조 필요)

### 종신보험
- 정의: 피보험자가 사망할 때까지를 보험기간으로 하는 생명보험으로, 평생 사망 보장을 제공합니다.
- 분류: 보장
- 키워드: 평생 보장, 사망 보장, 생명보험, 보험료
- 출처: 출처 미확인 (블로그 출처 — 검수 시 공식 출처 대조 필요)

### 정기보험
- 정의: 가입할 때 정해놓은 보험기간 내에만 사망을 보장하는 생명보험으로, 기간 종료 후에는 보장이 끝납니다.
- 분류: 보장
- 키워드: 기간 제한, 사망 보장, 정해진 기간, 갱신
- 출처: 출처 미확인 (블로그 출처 — 검수 시 공식 출처 대조 필요)

### 특약
- 정의: 기본 보험계약(주계약)에 추가로 부가하여 보장 범위를 확대하는 선택적 보험 계약입니다.
- 분류: 보장
- 키워드: 부가 보험, 추가 보장, 선택, 주계약 부가
- 출처: https://fine.fss.or.kr/main/prc/is/sub/is004.jsp

### 주계약
- 정의: 보험계약의 기본이 되는 주요 보험 계약으로, 특약은 이에 부가되는 형태입니다.
- 분류: 보장
- 키워드: 기본 보험, 핵심 보장, 주요 계약, 필수
- 출처: https://fine.fss.or.kr/main/prc/is/sub/is004.jsp

### 갱신형
- 정의: 정해진 갱신 주기(1년, 3년, 5년, 10년 등)가 도래할 때마다 보험료를 재계산하고 계약을 갱신하는 보험 형태입니다.
- 분류: 보장
- 키워드: 주기적 갱신, 보험료 인상, 재계산, 연령 적용
- 출처: https://fine.fss.or.kr/main/prc/is/sub/is015.jsp

### 비갱신형
- 정의: 보험 계약 기간 동안 보험료가 고정되는 보험 형태로, 계약 종료 시까지 같은 금액의 보험료를 납입합니다.
- 분류: 보장
- 키워드: 고정 보험료, 보험료 인상 없음, 장기 고정
- 출처: 출처 미확인 (블로그 출처 — 검수 시 공식 출처 대조 필요)

### 면책기간
- 정의: 보험계약이 시작된 후 보험사가 보장 책임을 지지 않는 일정 기간을 말합니다.
- 분류: 보장
- 키워드: 대기 기간, 보장 제외, 계약 개시, 조건
- 출처: https://fine.fss.or.kr/main/prc/is/sub/is015.jsp

### 감액기간
- 정의: 보험계약 개시 후 일정 기간 동안 보험금을 감액하여 지급하는 기간을 말합니다.
- 분류: 보장
- 키워드: 보험금 감액, 일정 기간, 축소 지급
- 출처: 출처 미확인

### 보장개시일
- 정의: 보험계약에서 보험회사가 보장 책임을 시작하는 날짜로, 이 날부터 보험사고가 발생할 때 보험금을 지급합니다.
- 분류: 보장
- 키워드: 보장 시작일, 효력 발생, 책임 개시
- 출처: https://fine.fss.or.kr/main/prc/is/sub/is004.jsp

### 계약자
- 정의: 보험회사와 보험계약을 체결하고 보험료를 납입하는 사람입니다.
- 분류: 보장
- 키워드: 계약 당사자, 보험료 납입자, 가입자
- 출처: 출처 미확인 (보험사 페이지 — 검수 시 공식 출처 대조 / ※ 계약 카테고리 중복 가능)

### 피보험자
- 정의: 보험사고의 기준이 되는 사람으로, 생명보험에서는 생사가 보험사고 발생의 객체가 되는 사람입니다.
- 분류: 보장
- 키워드: 보험 대상자, 피해자, 보험사고 기준
- 출처: 출처 미확인 (보험사 페이지 — 검수 시 공식 출처 대조 / ※ 계약 카테고리 중복 가능)

### 보험수익자
- 정의: 보험금을 받을 권리를 가진 사람으로, 계약자와 피보험자와 다를 수 있습니다.
- 분류: 보장
- 키워드: 보험금 수령자, 수익자 지정, 당사자
- 출처: 출처 미확인 (보험사 페이지 — 검수 시 공식 출처 대조 / ※ 계약 카테고리 중복 가능)

### 보험기간
- 정의: 보험회사(보험자)의 책임이 시작되어 끝날 때까지의 기간으로, 책임기간 또는 위험기간이라고도 합니다.
- 분류: 보장
- 키워드: 계약 기간, 책임 기간, 보장 기간, 위험 기간
- 출처: 출처 미확인 (보험사 페이지 — 검수 시 공식 출처 대조 필요)

### 보험료
- 정의: 보험 계약에서 계약자가 보험회사에 지불하는 금액입니다.
- 분류: 보장
- 키워드: 월 납입, 연 납입, 보험료 지급, 계약비용
- 출처: https://fine.fss.or.kr/main/prc/is/sub/is004.jsp

### 약관
- 정의: 보험계약의 내용과 범위, 보험금 지급 조건, 지급 제한 사유 등을 정해놓은 약정된 조건을 말합니다.
- 분류: 보장
- 키워드: 계약 조건, 보험 조건, 규정, 계약서
- 출처: 출처 미확인 (보험사 페이지 — 검수 시 공식 출처 대조 필요)

### 해지환급금
- 정의: 보험 계약자가 보험을 중도에 해지할 때 보험회사로부터 돌려받는 금액입니다.
- 분류: 보장
- 키워드: 중도해지, 환급, 보험료 반환
- 출처: 출처 미확인 (블로그 출처 — 검수 시 공식 출처 대조 / ※ 제도 카테고리 중복 가능)

### 납입면제
- 정의: 특정 사유(상해 80% 이상 후유장해, 암·뇌졸중·급성심근경색 진단 등)가 발생할 때 다음 보험료 납입을 면제하는 보험 제도입니다.
- 분류: 보장
- 키워드: 보험료 면제, 중대질병, 장해, 특정 조건
- 출처: 출처 미확인 (블로그 출처 — 검수 시 공식 출처 대조 필요)

### 암진단비
- 정의: 암으로 진단받았을 때 치료비 부담을 돕기 위해 지급되는 보험금입니다.
- 분류: 보장
- 키워드: 암 진단, 진단금, 의료비 보조, 중대질병
- 출처: 출처 미확인 (블로그 출처 — 검수 시 공식 출처 대조 필요)

### 뇌졸중진단비
- 정의: 뇌졸중(뇌출혈, 뇌경색 등)으로 진단받았을 때 지급되는 보험금으로, 3대 질병 중 하나입니다.
- 분류: 보장
- 키워드: 뇌혈관질환, 뇌질환, 진단금, 중대질병

### 급성심근경색진단비
- 정의: 급성심근경색증으로 진단받았을 때 지급되는 보험금으로, 3대 질병 보장에 포함됩니다.
- 분류: 보장
- 키워드: 심장질환, 급성질환, 진단금, 심근경색
- 출처: 출처 미확인 (검수 시 공식 출처 대조 필요)

### 고지의무
- 정의: 보험계약자나 피보험자가 보험 계약 체결 시 과거 병력, 직업, 거주지 등 중요한 사항을 보험회사에 알려야 할 법적 의무입니다.

- 키워드: 고지, 알릴 의무, 병력 고지, 계약 조건

### 자살면책
- 정의: 보험계약 개시 후 일정 기간(일반적으로 2년) 내에 피보험자가 자살할 경우 보험금을 지급하지 않는 약관 조항입니다.
- 분류: 보장
- 키워드: 자살 제외, 면책 기간, 보장 제외, 계약 조건

### 배당
- 정의: 보험회사가 보험료 운용 차익에서 일정 금액을 계약자에게 돌려주는 제도입니다.
- 분류: 보장
