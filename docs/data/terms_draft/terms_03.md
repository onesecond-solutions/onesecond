# terms_03 — 인수/언더라이팅 (채굴 직원 3, 1차 사이클)

### 우량체
- 정의: 건강 상태가 좋고 질병이나 장해가 없어 보험사가 평가한 위험도가 낮은 피보험자. 일반적으로 비흡연, 정상 BMI, 정상 혈압, 의료 병력이 없는 경우를 말함.
- 분류: 인수
- 키워드: 선호 건강체, 우수 위험도, 보험료 할인, 저위험군
- 출처: https://www.samsunglife.com/individual/display/customerservice/PDP-CCUSE010110M/107575

### 표준체
- 정의: 평균적인 건강 상태를 가진 피보험자로, 우량체보다는 위험이 높지만 거절체보다는 낮은 중간 등급의 피보험자.
- 분류: 인수
- 키워드: 표준 위험도, 일반 건강체, 평균 보험료, 중간 등급
- 출처: https://edu.knia.or.kr/edu/knowledge2.do

### 표준미달체
- 정의: 우량체보다 위험이 높은 피보험자로, 만성질환, 치료 이력, 부담보 대상 질병이 있어 표준 보험료 이상의 할증이 적용될 수 있는 등급.
- 분류: 인수
- 키워드: 초과 위험도, 비표준체, 할증 대상, 고위험군
- 출처: https://www.fcsc.kr/C/fu_c_03_02.jsp?faq_seq=8331&lineNo=8

### 거절체
- 정의: 보험사의 인수심사 결과 사망 확률이 높거나 위험도가 과도하여 보험 계약을 거절하기로 판단된 피보험자.
- 분류: 인수
- 키워드: 부승인, 인수 거절, 사절, 비승인
- 출처: https://www.fcsc.kr/C/fu_c_03_02.jsp?faq_seq=8331&lineNo=8

### 연기
- 정의: 보험사가 인수심사 중 추가 정보나 조사가 필요할 때 보험금 지급을 일시적으로 보류하거나 청약 승인을 미루는 인수 결과.
- 분류: 인수
- 키워드: 보류, 재심사, 추가 조사, 지급 연기
- 출처: https://www.fcsc.kr/C/fu_c_03_02.jsp?faq_seq=8331&lineNo=8

### 언더라이팅
- 정의: 보험 청약자의 신체적, 환경적, 도덕적, 재정적 위험을 평가하여 보험계약 인수 여부, 할증/할인 여부, 부담보 조건 등을 종합 판단하는 보험사의 의사결정 과정.
- 분류: 인수
- 키워드: 인수심사, 위험 평가, 계약 승인, 종합 판단
- 출처: https://www.fcsc.kr/C/fu_c_03_02.jsp?faq_seq=8331&lineNo=8

### 인수심사
- 정의: 청약자의 청약 내용이 성실하게 고지되었는지, 위험도가 적절한지를 객관적·주관적으로 평가하여 보험계약의 인수 여부를 결정하는 절차.
- 분류: 인수
- 키워드: 위험심사, 계약심사, 신청 검토, 승인 판정
- 출처: https://edu.knia.or.kr/edu/step2_1.do

### 계약적부조사
- 정의: 보험 계약의 유효성과 정당성을 확인하기 위해 보험사가 수행하는 조사로, 고지내용, 건강상태, 재정력 등을 검증하는 과정.
- 분류: 인수
- 키워드: 적부 판정, 계약 검증, 유효성 확인, 합법성 검토
- 출처: https://www.knia.or.kr/file-manager/102857

### 할증
- 정의: 피보험자의 위험도가 표준보다 높을 때 기본 보험료에 일정 비율(예: 10%, 20%)을 더 가산하여 계산한 보험료 인상.
- 분류: 인수
- 키워드: 보험료 인상, 위험할증, 추가 보험료, 보험료 조정
- 출처: https://kbthink.com/insurance-trend/occupational-grade.html

### 할인
- 정의: 피보험자의 위험도가 표준보다 낮을 때 기본 보험료에서 일정 비율을 감소시켜 계산한 보험료 인하.
- 분류: 인수
- 키워드: 보험료 인하, 위험할인, 우량체할인, 무사고할인
- 출처: https://www.shinhanlife.co.kr/hp/cdhi0360t01.do

### 부담보
- 정의: 특정 질병이나 신체 부위에 대해 일정 기간(보통 1~5년) 보험사의 보장 책임을 제한하거나 제외하는 인수 조건.
- 분류: 인수
- 키워드: 담보 제외, 특정부위 제외, 조건부 담보, 제한 담보
- 출처: https://www.fins.co.kr/news/articleView.html?idxno=102084

### 특정부위 부담보
- 정의: 피보험자의 과거 병력이나 치료 이력이 있는 특정 신체 부위나 질병에 대해 보험금을 지급하지 않거나 제한적으로 지급하도록 설정하는 조건.
- 분류: 인수
- 키워드: 부위 제외, 부분 담보 제외, 조건부 담보, 부담보 조건
- 출처: https://www.kbanker.co.kr/news/articleView.html?idxno=90091

### 면책
- 정의: 보험약관에서 명시한 특정 사건이나 조건이 발생했을 때 보험사가 보험금을 지급할 책임이 없는 항목.
- 분류: 인수
- 키워드: 지급 거절, 보장 제외, 책임 면제, 불담보
- 출처: https://www.insunet.co.kr/insurance-terms/1016

### 면책기간
- 정의: 부담보가 적용되는 기간으로, 일반적으로 1~5년 범위에서 보험사의 인수심사 결과에 따라 정해지며, 해당 기간 동안 부담보 부위의 질병 진단을 받지 않으면 이후 보험금을 청구할 수 있음.
- 분류: 인수
- 키워드: 부담보 기간, 제외 기간, 조건부 담보 기간, 대기 기간
- 출처: https://www.kbanker.co.kr/news/articleView.html?idxno=90091

### 고지의무
- 정의: 보험 계약 체결 시 청약자나 피보험자가 보험사에게 중요한 사실(건강 상태, 병력, 직업 등)을 성실하게 고지해야 할 법적 의무.
- 분류: 인수
- 키워드: 공시 의무, 정보 제공, 사실 고지, 신고 의무
- 출처: https://kbthink.com/insurance-guide/notification.html

### 고지의무위반
- 정의: 보험 가입 시 보험사의 질문에 대해 사실과 다르게 답하거나 중요한 정보를 의도적으로 숨기는 행위로, 보험금 지급 거절이나 계약 해지의 사유가 될 수 있음.
- 분류: 인수
- 키워드: 허위 고지, 정보 은폐, 조건 위반, 계약 위반
- 출처: https://www.gardenbom.com/31428/insurance-duty-disclosure-claim/

### 도덕적위험
- 정의: 보험 계약 체결 후 피보험자가 주의의무를 소홀히 하여 사고 위험이 증가하는 현상. 정보 비대칭으로 인해 보험사가 통제하기 어려운 피보험자의 행동 변화.
- 분류: 인수
- 키워드: 보험 도덕, 부주의, 행동 위험, 정보 비대칭
- 출처: https://eiec.kdi.re.kr/material/echannView.do?idx=530&type=00002000240000100006

### 역선택
- 정의: 보험 가입 전부터 존재하는 정보의 비대칭으로 인해, 실제 위험도가 높은 사람이 낮다고 허위 고지하거나 낮은 위험도 상품에 가입하려는 선택 현상.
- 분류: 인수
- 키워드: 정보 비대칭, 역 선택, 우량한 위험만 남김, 위험의 선택
- 출처: https://eiec.kdi.re.kr/material/echannView.do?idx=530&type=00002000240000100006

### BMI
- 정의: 체질량 지수(Body Mass Index)로 체중(kg)을 키(m)의 제곱으로 나눈 값. 보험 인수 시 비만도를 평가하는 주요 지표로, 일반적으로 18.5~23 사이를 정상 범위로 봄.
- 분류: 인수
- 키워드: 체질량 지수, 비만도, 위험 평가 지표, 건강 지표
- 출처: https://ko.wikipedia.org/wiki/%EC%B2%B4%EC%A7%88%EB%9F%89_%EC%A7%80%EC%88%98

### 직업급수
- 정의: 직업의 위험도를 분류한 등급으로, 보험사가 직업별 사고 통계와 보험금 지급 이력을 바탕으로 1급(저위험)부터 3급(고위험)으로 구분하여 보험료를 산정함.
- 분류: 인수
- 키워드: 직업 분류, 위험 등급, 직업 위험도, 보험료 등급
- 출처: https://kbthink.com/insurance-trend/occupational-grade.html

### 건강진단
- 정의: 보험 인수 과정에서 청약자의 건강 상태를 의학적으로 확인하기 위해 수행하는 검사로, 혈압, 혈액 검사, 신체 측정 등을 포함.
- 분류: 인수
- 키워드: 의료검진, 건강검사, 인수 검진, 위험 평가 검사
- 출처: 출처 미확인

### 문진
- 정의: 보험 청약 시 청약자의 건강 상태, 병력, 약물 복용, 수술 경험 등을 기록된 질문지를 통해 조사하는 절차.
- 분류: 인수
- 키워드: 건강 설문, 병력 조사, 청약서 작성, 정보 수집
- 출처: https://kbthink.com/insurance-guide/notification.html

### 위험평가
- 정의: 보험사가 청약자의 신체적, 환경적, 도덕적, 재정적 위험을 종합 평가하여 그 수준을 판단하고 인수 조건을 결정하는 과정.
- 분류: 인수
- 키워드: 위험 심사, 위험 등급 판정, 종합 평가, 인수 판정
- 출처: https://edu.knia.or.kr/edu/step2_1.do

### 할증보험료
- 정의: 청약자의 위험도가 표준을 초과할 때 기본 보험료에 할증률을 곱한 금액으로, 위험이 높을수록 할증율이 커짐.
- 분류: 인수
- 키워드: 높은 보험료, 추가 보험료, 위험 할증, 보험료 인상액
- 출처: https://kbthink.com/insurance-trend/occupational-grade.html

### 할인보험료
- 정의: 청약자의 위험도가 표준보다 낮을 때 기본 보험료에서 할인율을 곱하여 감소시킨 금액.
- 분류: 인수
- 키워드: 낮은 보험료, 감액, 우량체할인, 무사고할인
- 출처: https://www.shinhanlife.co.kr/hp/cdhi0360t01.do

### 감액
- 정의: 보험 계약을 유지하면서 보장 금액(보험금)을 줄이는 것으로, 보장 금액 감소에 따라 보험료도 비례하여 감소하는 보험료 조정 방식.
- 분류: 인수
- 키워드: 보장금액 감소, 보험료 조정, 부분 보장, 보험금 감소
- 출처: https://blog.signalplanner.co.kr/587/

### 인수결과
- 정의: 보험 청약에 대한 인수심사를 완료한 후 보험사가 내린 최종 결정으로, 승인, 조건부 승인(할증, 부담보), 연기, 거절 중 하나.
- 분류: 인수
- 키워드: 심사 결과, 승인 판정, 계약 판정, 인수 판정
- 출처: https://edu.knia.or.kr/edu/step2_1.do

### 청약서
- 정의: 보험 계약을 신청할 때 청약자가 작성하는 문서로, 개인 정보, 건강 상태, 병력, 직업 등 인수심사에 필요한 정보를 기록함.
- 분류: 인수
- 키워드: 가입 신청서, 계약 청약, 신청 문서, 고지 서류
- 출처: https://kbthink.com/insurance-guide/notification.html

### 특약
- 정의: 기본 보험(주계약)에 추가하여 가입할 수 있는 선택적 보장으로, 암 진단 특약, 입원 특약 등 다양한 형태가 있음.
- 분류: 인수
- 키워드: 추가 담보, 선택 보장, 부수 계약, 보장 옵션
- 출처: 출처 미확인

### 조건부승인
- 정의: 보험 청약을 승인하되, 할증, 부담보, 특정 조건 추가 등의 제약을 두고 승인하는 인수 결과.
- 분류: 인수
- 키워드: 조건부 가입, 제약 승인, 조건 부담보, 제한 승인
- 출처: https://edu.knia.or.kr/edu/step2_1.do

### 신체적위험
- 정의: 청약자의 신체 건강 상태와 관련된 위험으로, 현재 질병, 과거 병력, 유전 질환, 장해 등이 포함됨.
- 분류: 인수
- 키워드: 건강 위험, 신체 상태, 질병 위험, 건강 문제
- 출처: https://edu.knia.or.kr/edu/step2_1.do

### 재정적위험
- 정의: 청약자의 경제적 상황과 관련된 위험으로, 보험 가입 능력 부족이나 과도한 보험금 청구 유인이 있는지를 평가하는 요소.
- 분류: 인수
- 키워드: 경제 상황, 납입 능력, 과다 보험 위험, 재정 신용도
- 출처: https://edu.knia.or.kr/edu/step2_1.do

### 환경적위험
- 정의: 청약자의 생활 환경이나 직업 환경으로 인한 위험으로, 직업의 위험도, 거주 지역, 작업 환경 등이 포함됨.
- 분류: 인수
- 키워드: 직업 위험, 작업 환경, 거주 환경, 외부 위험
- 출처: https://edu.knia.or.kr/edu/step2_1.do

### 사망률
- 정의: 특정 집단에서 일정 기간 동안 사망 가능성을 백분율로 나타낸 지표로, 보험사의 인수심사 시 위험도 평가의 핵심 기준.
- 분류: 인수
- 키워드: 생명 위험, 사망 확률, 위험도 지표, 생사 위험
- 출처: 출처 미확인

### 소급면책
- 정의: 보험 계약 체결 전에 발생한 사건이나 질병에 대해 보험금을 지급하지 않는 조건.
- 분류: 인수
