## 주요 수정 권장 (우선순위)

### 1순위 — 용어 오류 (즉시 수정)
- **대위변제** → 용어명 "보험자대위"로 변경 필수. 법률 오용.
- **손해사정사** → 출처 나무위키 → law.go.kr 보험업법 제185~186조로 교체. 감독기관 표현 정정("금융감독원 지도·감독" → "금융위원회 등록, 금감원장 시험 실시").

### 2순위 — 비공식 출처 교체 (신뢰도)
- **소멸시효** — casenote.kr → law.go.kr 상법 제662조
- **비례보상** — insunet.co.kr → easylaw 손해보험 섹션 + 상법 제672조
- **배당** — newsis.com → fine.fss.or.kr 또는 klia.or.kr
- **역선택** — fastercapital.com → easylaw 또는 klia.or.kr
- **보험가액·피보험이익** — ko.wikipedia.org → law.go.kr 상법
- **고지의무** — kbthink.com → easylaw + 상법 제651조
- **감액기간·특약** — kbthink.com → fine.fss.or.kr 또는 표준약관
- **보험약관** — hanalife.co.kr → fine.fss.or.kr
- **피보험자·보험계약자·보험수익자·보험료·보험료 납입** — cardif.co.kr → easylaw 또는 fine.fss.or.kr

### 3순위 — 중복 정리
- **보험 청약철회** vs **철회** — 동일 개념 중복 수록. 1개 삭제 또는 통합 검토.
- **피보험자·보험계약자·보험수익자·고지의무·청약·보험료·보험약관·특약** — 계약 배치와 중복 여부 배치 간 대조 후 결정.

### 4순위 — 정의 보완
- **보험사고** — 손·생보 통합 표현으로 수정
- **피보험자** — 생명보험/손해보험 피보험자 구분 표기 보완
