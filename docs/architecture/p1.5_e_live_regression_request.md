# § 5. 사고 신호 발생 시 즉시 정지

다음 신호가 보이면 시연 중단하고 Code에 raw 회신:
- **500 에러** (signup endpoint 또는 token endpoint)
- **23503 FK 위반** (branches/teams/insurers 정합 깨짐 — branches RLS 처방 회귀 의심)
- **400 에러** (raw_user_meta_data 14키 결손 또는 JSON 형식 오류)
- **로그인 후 app.html 진입 실패** (`../app.html` 상대 경로 미정합)
- **Google 버튼 활성** (disabled가 빠진 경우 — Phase 2 미준비 상태에서 활성 위험)
- **모달 미열림** (`openAuthModal is not defined` 또는 hidden 안 풀림)

→ Code가 진단 후 후속 commit. 본 시연 결과 raw가 진실 원천.
