## 4. 하·확인불가 (별 트랙)
- [하] **재정적 언더라이팅(Financial Underwriting)** — 원문 body의 '합산 사망담보 30억 원 초과 시 재정서류 제출 요구'는 사실 오류. 실제 가이드라인의 특별재정심사 발동 조건은 단순 금액 초과가 아니라 3가지 조건(계약자≠피보험자, 사망담보 중복 계약, 30억 이상+4건 이상) 동시 충족이어야 함. 검색 결과(KIRI 보험법 동향 15호, 서울경제 보도 기반 가이드라인 내용)로 확인. Body를 정확한 3조건 동시 충족 요건으로 교정.
- [하] **승낙 의제(Deemed Acceptance) — 심사 30일 초과 시** — 원문 body에서 '진단계약은 진단일을 기준으로 30일이 적용된다'는 표현이 부정확. 상법 638조의2 조문상 정확한 표현은 '인보험계약의 피보험자가 신체검사를 받아야 하는 경우 그 기산점은 신체검사를 받은 날'이다. '진단계약'이라는 특정 용어와 '진단일' 기산은 조문 원문과 다소 다른 표현. 교정본으로 대체.
- [확인불가] **직업 위험 소급 추징 분쟁** — 원 source_title에 명시된 '금융감독원 보도참고자료(2022.9.23) 직무변경 통지의무 관련 소비자 유의사항'은 복수 검색에서 직접 확인 불가. '보험저널 팩트체크'는 비공식 언론 출처. 소급 추징 분쟁 자체는 실무에서 발생하는 사실이나 지정된 날짜의 금감원 보도자료를 공식 출처로 확정할 수 없어 '확인불가' 판정.
