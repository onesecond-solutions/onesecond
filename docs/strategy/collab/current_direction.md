## 🔑 브랜드 정체성 (0순위, 모든 결정의 기준) ⭐⭐⭐

**중립 독립 SaaS** — 원세컨드는 어느 보험회사·GA(대리점)에도 소속되지 않은 독립 SaaS이며,
모든 사업 결정은 특정 회사 이해관계가 아닌 **"현장 전체 기준"의 중립 위치**에서 내린다.

**왜 독립성이 중요한가:**
- 보험 현장 = 다양한 보험사·GA 이해관계 충돌하는 영역
- 특정 회사 종속 시 → 그 회사 영업 도구로 전락 → 다른 회사 설계사 거부
- 원세컨드 = 모든 보험 현장이 사용 가능한 **운영 OS**

**오해 방지 (4팀 = 첫 사용자, 단독 고객 X):**
- 4팀(AZ 더원 GA의 한 팀) = 5/15 첫 사용자 검증
- 영구 단독 고객 X / AZ 종속 X
- 향후 다른 GA / 다른 보험사 매니저 입점 시 동일 OS

회귀 신호 (즉시 보고): ❌ 특정 보험사 기능 중심 / ❌ 특정 GA 편향 UX / ❌ 폐쇄형 조직 SaaS 해석
