## 목적
원세컨드 UX v2 표준 확정. 현재 보기=우측 패널 / 작성=모달 / 수정=모달 구조를 → **보기·작성·수정 전부 동일 작업공간(우측 패널)에서 처리**하도록 통일.

## 범위
이번 Phase 0 = **"작업공간 구조"만 구축.**

### 절대 포함 금지
- 리치 에디터 / 이미지 본문 삽입 / 표 개선 / 저장 구조 변경 / DB 변경

## 구현
### 공용 인라인 패널
- 기준 = `dia-panel`
- 공용 상태: `view` / `create` / `edit`
- 상태 라벨: 보기 / 작성 중 / 수정 중
- 상태 전환 시 시각적 차이 제공 (예: 상단 컬러 바 또는 패널 테두리 강조)
- **메모 UI와 혼동되는 노란 메모지 스타일 금지.**

### 글쓰기 버튼 표준
- 위치: 목록 상단 필터줄 기준 우측 작업공간 중앙
- 규격: 폭 약 210px · 높이 약 44px · accent token
- 동작: 글쓰기 → 좌측 임시 카드 생성 → 우측 create 상태 → 저장 → view 상태
- 취소 → 임시 카드 제거

### 모바일
- 기존 `show-viewer` 재사용. 신규 모바일 UX 금지.
- 목록 → 상세 → 작성 → 수정, Push 방식 유지.
