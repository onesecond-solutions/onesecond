# [작업지시서] Phase 1 — 작성공간(에디터) 고도화

> 작성: 기획실장(Web) / 승인: 대표 / 저장: 총괄실장(Code) / 일자: 2026-06-11
> **선행 조건: Phase 0 완료 후 착수. Phase 0 완료 전 작업 금지.**

## 목적
Phase 0 완료 후, 작성 패널을 단순 입력폼 → **실제 작업공간으로 확장.**

## 원칙
작성 화면은 입력창이 아니다. **작업공간이다.**
사용자는 동일 화면에서 작성·수정·이미지 삽입·표 작성·딸깍·카드 생성·요약을 수행 가능해야 함.

## 현재 상태
- 이미지 삽입 = 파일 첨부 (잘못 구현)
- 대표 의도 = **본문 내부 삽입** (예: 본문 / [이미지] / 본문)

## 이미지 삽입
- 파일 첨부 아님. **본문 내부 삽입.**
- 이미지는 Storage 저장 후 본문 내 참조.
- **§4 개인정보 정책 검토 필수.**

## 표 삽입
- 기존 기능 개선. 목표: 행 추가 / 열 추가 / 수정 / 정렬 가능.
- 텍스트 표 수준 금지.

## 편집 툴바
- 작성 화면 상단 배치 (예: 굵게 / 기울임 / 색상 / 정렬 / 이미지 / 표)
- 현재 구현된 이미지·표·캘린더 도구 재활용 가능.

## 적용 범위
보험사 자료실 / 보험 Q&A 질문 / 오피스 / MY SPACE / 공동 스페이스 / 함께해요
