## 1. 작업 범위 (4단계, 4/25 합의 — 그대로 보존)

### Step 1 — 검색 최적화 (전체 56개 또는 현재 행 수)

| 항목 | 내용 |
|---|---|
| 대상 컬럼 | `title`, `highlight_text` 만 |
| 절대 금지 | `script_text` 본문 수정 |
| 목적 | A2 검색창에서 키워드(질병명·연령·상황어) 노출 |
| 톤 | 팀장님 17년 현장 톤 / 자연스러운 한국어 |
| 형식 | "기존 제목 — 검색 키워드 추가" 패턴 |

### Step 2 — 실명 익명화 (id 5, 13, 21)

| 항목 | 내용 |
|---|---|
| 대상 | `script_text` 본문에서 실명만 제거 |
| 제거 실명 (4/25 명시) | 김철민 / 강원래 / 이윤석 / 이병헌 등 |
| 익명화 표기 | "○○○ 씨" / "한 연예인이" 형태 |
| 사유 | 보험업법·퍼블리시티권 위험 차단 |

### Step 3 — 사례·통계 보강 (need_emphasis 11개)

| 항목 | 내용 |
|---|---|
| 대상 id (need_emphasis) | 9, 19, 20, 28, 44, 58 (6개) |
| 대상 id (need_emphasis_2) | 26, 31, 37, 43, 51 (5개) |
| 대상 컬럼 | `script_text` |
| 보강 내용 | 연예인 사례(익명) + 통계(출처 명기) |
| **⭐ 도표 처리** | **빼고 텍스트로만** (4/25 팀장님 명시) |
| 도표 추가 | 시간 여유 시만 허용 (강요 X) |
