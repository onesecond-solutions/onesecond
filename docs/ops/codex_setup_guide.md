> **전략:** Plus($20)로 **세팅+파일럿 검증 먼저** → 작동 확인되면 Pro($100)로 올려 대량 가동. 비용 먼저 지르지 않는다.
> **대표님이 할 일 = 이 순서대로 클릭. 막히면 막힌 화면을 그대로 알려주세요. 총괄팀장이 풉니다.**

## 0. 전체 그림 (1분)

- 코덱스 = OpenAI판 코딩 에이전트(총괄팀장 Code의 OpenAI 버전).
- 우리는 **코덱스 클라우드**를 쓴다 = 대표님 PC 안 켜도 OpenAI 클라우드에서 일하고 결과를 **PR로 제출**.
- 코덱스는 **데이터 준비만** 한다(채굴·PDF OCR·DB 적재). 제품 화면은 총괄팀장이 만든다.
- 코덱스는 repo 안 `AGENTS.md` + 총괄팀장이 준 작업지시서만 따른다.

## 1. 코덱스 켜기 (Plus 구독에 포함)

1. 브라우저에서 **https://chatgpt.com/codex** 접속 (대표님 ChatGPT Plus 계정으로 로그인).
2. Plus 요금제에 코덱스가 **이미 포함**돼 있음 — 추가 결제 없이 진입됩니다.

> Pro 변경은 나중에. 지금은 Plus 그대로.

1. 코덱스가 "Connect GitHub / GitHub 연결"을 요청 → 승인.
2. 저장소 목록에서 **`onesecond-solutions/onesecond`** 선택(권한 허용).
3. 연결되면 코덱스가 우리 코드를 읽고 PR을 만들 수 있게 됩니다.

> ⚠️ 권한 범위를 묻거든 `onesecond` 저장소만 선택해도 됩니다(전체 권한 줄 필요 없음).

## 5. 첫 작업(파일럿) 던지기

1. 코덱스 화면에서 **"새 작업 / New task"** 입력칸에, 총괄팀장이 드린 **작업지시서(`docs/work_orders/`)의 내용을 그대로 붙여넣기.**
2. "이 지시서만 따르라"는 문구가 지시서 맨 위에 있습니다.
3. 코덱스가 클라우드에서 작업 → 끝나면 **PR**을 만듭니다.
4. PR이 생기면 **총괄팀장에게 알려주세요** — 제가 검수하고 머지 여부를 판단합니다.

  - Pro = 클라우드 병렬 작업 = 여러 채굴·OCR 동시 = 24시간 대량 가동.
- 그 전까지는 Plus로 검증만.
