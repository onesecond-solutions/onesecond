'넓은 문'인지

이름은 비슷해도 실제 보장 범위가 달라요. 얼마나 넓게 담는 구조인지가 핵심입니다.

왜 중요한가요
발생하면 부담이 큰 만큼, '범위'가 핵심
뇌·심장은 진단금 액수도 중요하지만, 어떤 병까지 보장되는지 범위가 더 결정적이에요. 범위가 좁으면 정작 필요할 때 못 받을 수 있어요.
이렇게 봐요
'좁은 담보'인지 '넓은 담보'인지
같은 뇌·심장이라도 담는 범위가 이렇게 달라요.
