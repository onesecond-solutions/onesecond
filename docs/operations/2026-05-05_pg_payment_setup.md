- 팀장님이 "가입 인증을 문자로 했던 것 같다"고 회상하셨으나
- 확인 결과 = **Resend는 이메일 발송 서비스** (SMS 아님)
- 용도: Supabase Auth ↔ Resend 연동으로 `noreply@onesecond.solutions` 도메인 메일 발송
