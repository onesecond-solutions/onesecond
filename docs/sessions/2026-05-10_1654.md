### 5/10 오후 본 세션 사고 — `*/` 토큰 multi-line 주석 조기 종료 ⭐⭐⭐

- **본질:** board.html `/* 모든 사용자 (admin/ga_*/insurer_*) */` 주석 안 `_*/` 패턴 → JavaScript multi-line 주석 종료 토큰 형성 → 주석 조기 종료 → 그 다음 `insurer_*) */` 코드로 해석되어 SyntaxError → IIFE 전체 실행 실패 → window.* 30+ 함수 등록 실패
- **단순 문법 오류 아님 — Code 자체 검증 게이트 약함:**
  - commit 직전 syntax 자가 검증 안 돌렸음 (라이브 사고 후 발견)
  - node -e "new Function(...)" 1줄로 발견 가능했음
- **처방:**
  - ① JavaScript multi-line 주석 안 `_*/` 또는 `*/` 토큰 박지 말 것 (정규식 슬래시 + 별표 조합 등에서 발생)
  - ② commit 직전 syntax 자가 검증 표준: `node -e "new Function(scriptBlock)"` 1줄로 발견 가능
  - ③ 본 패턴은 board.html 같은 inline `<script>` 영역에서 특히 위험 (HTML fragment loader가 IIFE로 다시 래핑)
