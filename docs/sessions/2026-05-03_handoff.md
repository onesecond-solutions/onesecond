## 1. 본 세션 핵심 결정 사항

### 1-1. 자료 자산화 트랙이 메인 트랙으로 격상

기존 "스크립트 보강 프로젝트"는 작은 그림이었음. 본 세션에서 큰 그림 재정의:

- 단순 스크립트 보강 → **자료 자산화 + 5개 멀티 유즈 통합 트랙**
- One Source Multi Use 구조 확정
- v4.0 보험판 구글의 첫 실체화

### 1-2. 5개 멀티 유즈 (One Source Multi Use)

자료 자산화 한 번 하면 다섯 갈래로 활용:

1. 스크립트 자동 증식
2. 검색 엔진 (특약명 매핑)
3. 게시판 자동 노출
4. AI 응답 (v4.0 학습 기반)
5. 광고 매칭 (맥락 기반)

### 1-3. 정면 돌파 전략 합의 — 단, 보류 결정 (5/3)

**합의된 전략:**
- 무허가 DB화 + 검색 노출 → 항의 시점 협상 카드 활용
- 원수사 항의 시 → 원본 내림 + 재가공 검색 결과 유지 + 해당 회사 입점 차단

**식별된 위험:**
- 금융소비자보호법 제22조 영역 (일반 저작권보다 강함)
- 메리츠 PDF: "고객 교부 및 온라인게시 목적으로 사용 할 수 없습니다"
- NH PDF: "교부목적으로 사용할 수 없으며, 임의로 편집 또는 수정할 수 없습니다"
- 대형사 소송 + 손해배상 가능성
- "재가공 저작권 = 원세컨드"는 법정 다툼 여지 있음

**결정:** 변호사 자문 1회 받기 전 DB화 진행 보류

### 1-4. Desktop = A. 자동 처리 파이프라인 역할

```
[별도 PC]
  ├─ Claude Desktop (대화 인터페이스, MCP 도구)
  ├─ Make (자동화 워크플로우)
```

## 5. AI 작업 실패 학습 (5/3 메리츠 PDF)

### 5-1. 추론 금지의 정의 재정립

**잘못된 추론 (절대 금지):**
- 숫자 변형 ("31,636원" → "약 3만 원대")
- 데이터 왜곡 ("70%" → "대부분")
- 출처 없는 통계 끼워 넣기

**필요한 작업 (추론 아님):**
- PDF 안 단어 매핑 ("2대.치.단.비" = "2대 치료비까지 주는 진단비")
- 정보 재구조화 ("메리츠 31,636원, B사 36,390원" → 비교표)
- 표로 모으기

→ AI에게 "추론 금지" 대신 **"PDF 안 단어만 사용 + 재구조화 OK"**라고 명령해야 함

### 5-2. 한 번에 처리 단위

- 26페이지 한 번에 처리 = 실패
- 페이지 또는 페이지 묶음 단위로 분할 = 성공
- 메리츠 26페이지 = 약 8개 작업 단위

### 5-3. PDF별 난이도

- **NH:** 텍스트 + 도표 분리, 처리 쉬움
- **메리츠:** 이미지 안 글자 박힘 + 색상 강조 의미 전달, 처리 어려움
- **자체 약어 있는 회사:** 매핑표 먼저 만들고 본문 작업 시작

### 5-4. 메모리 등록 권장 사항

다음 세션에서 메모리 추가 등록 권장:

```
보험 PDF 작업 시 절대 원칙:
1. 페이지 또는 페이지 묶음 단위로 분할 처리 (한 번에 X)
2. 자체 약어 있는 회사는 매핑표 먼저 만들기
3. "PDF 안 단어만 사용 + 재구조화 OK"가 추론 금지의 정확한 정의
4. 숫자 변형·데이터 왜곡·출처 없는 통계는 절대 금지
5. 4종 산출물 패턴 (약어 매핑 + 회사 간 매핑 + TM 멘트 + 검색 메타데이터)
```

## 10. 본 세션에서 인지된 자료 자산화 트랙 핵심 통찰

### 10-1. 모든 멀티 유즈는 한 출처에서 나옴

```
4팀 단톡방 PDF (1년치)
    ↓
[자료 자산화]
    ↓
스크립트 / 검색 / 게시판 / AI / 광고
    ↓
설계사 사용 (v1.1 → v1.5 → v2.0)
    ↓
v2.0 원수사 입점 협상 카드
    ↓
원수사 직접 자료 등록 (피드백 루프)
    ↓
v4.0 보험판 구글
```

### 10-2. 진입 장벽 = 시간 자체

1년치 × 회사 10개 = 120건+ 구조화된 데이터.
경쟁사가 따라오려면 같은 시간 필요. **시간이 해자.**

### 10-3. v2.0 협상 멘트 변경

기존: "우리 게시판 쓰세요"
신규: **"귀사 1년치 자료가 우리 플랫폼에서 매일 검색됩니다. 직접 올리시면 더 정확하게 노출됩니다."**

### 10-4. AI 학습 데이터 차별화

- ChatGPT/Claude 일반: 보험 일반 지식 (소비자 마케팅 글 + 약관)
- 원세컨드 v4.0: **회사별 특약명 매핑 + 월별 트렌드 + TM 현장 멘트 변환**
