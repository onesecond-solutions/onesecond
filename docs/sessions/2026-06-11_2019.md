## 🚨 학습 / 결정
- **display:'' ≠ display:none 해제** — inline `style.display=''`는 inline 속성만 제거, id 셀렉터 `display:none` CSS 규칙은 못 덮음 → `'block'` 명시 필요. (#571·#572 두 곳 동일 함정)
- **그리드 칸 오버플로우** — 그리드 아이템 기본 min-width:auto + 넓은 콘텐츠(표·이미지) → 칸이 트랙 밖으로 확장. `min-width:0`(아이템) + `overflow:auto`(내부 스크롤)가 표준 해법. text-align:center 요소가 우측으로 밀리면 부모 칸 폭 확장을 의심.
- **보기 렌더 img 제약 누락** — 에디터(.osrich-body)엔 img max-width:100% 있어도 저장 본문 보기 렌더엔 별도로 필요. 보기/편집 양쪽 다 제약해야 함.
- **칩 라벨 변경 안전 조건** — onclick에 필터 인자 명시 전달(filterMys(this,'전체'))이면 표시 텍스트만 바꿔도 동작 불변. 단 textContent를 카테고리와 비교하는 로직 없는지 확인 후 변경.
