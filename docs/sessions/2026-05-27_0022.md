### 2. PR #106 — landing CTA 닫는 태그 + 헤더 로그인 outline (격차 1·2)

**팀장님 라이브 점검 격차 보고:**

**격차 1 (HTML 문법 오류):** `pages/landing.html` line 599 `<button class="lp-cta-btn">` 닫는 태그가 `</a>`로 들어감 (어제 PR #94 추정 격차). 브라우저가 두 button을 nesting으로 해석 → "이미 회원이신가요? 로그인" 텍스트가 부모 button 안 자식 button 형식 = 왼쪽 정렬 + 회색 박스 + 글자 뭉개짐.

**격차 2 (헤더 두 버튼 무게감 불정합):** 헤더 베이지 vs 가입하기 인디고 fill = "붕 뜸" 격차. 로그인 = 텍스트만 (옅음) / 가입하기 = 인디고 fill + shadow (진함) = 무게차 큼.

**가동:**
- line 599 `</a>` → `</button>` (1자리)
- `.lp-login-link` = 인디고 outline + pill로 정정 (가입하기 fill과 톤 통일)
  - color: text-secondary → var(--color-brand)
  - border: 0 → 1px solid var(--color-brand)
  - padding: 10px 14px → 9px 21px
  - border-radius: radius-sm → radius-pill
  - hover = 인디고 fill + white text
  - 모바일 padding 8px 17px

### 3. PR #107 — 헤더 버튼 v2 토큰 통일 + box-shadow 제거 + 단색 fill (격차 발생)

**팀장님 명시:** v2 토큰 기준 (`--ac` / `--ach`) 통일 + 가입하기 "겹친 한 겹" (box-shadow) 제거.

**raw 점검:**
- `.lp-signup-link` box-shadow `0 4px 12px rgba(99,102,241,0.24)` = "겹친 한 겹" 본질 (옅은 인디고 글로우, `:focus outline` 아님)
- 정정 자료: `var(--color-brand)` → `var(--ac)` / 그라데이션 → 단색 / box-shadow 제거 / hover `--ach`

**격차 발생 (본인 추정 격차):** 본인이 `var(--ac)` 참조로 변경했으나 **`landing.html :root`에 `--ac` 변수 정의 없음** + **`tokens-v2.css` 미로드** + **`<html>`에 `data-theme` 속성 없음** → `var(--ac)` 빈 값 처리 → 가입하기 배경 미적용 → 흰 글자만 보임.

→ **PR #107 진입 전 `--ac` 변수가 페이지에 정의돼 있는지 사전 점검 누락** = 본인 추정 격차 또 발생.
