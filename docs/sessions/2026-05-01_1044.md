### Supabase RLS 경고 대응
신규 테이블 생성 시 RLS 경고 발생 → "Run without RLS" 선택 금지. RLS 활성화 + service_role 전용 정책 추가가 정석.

```sql
ALTER TABLE [테이블명] ENABLE ROW LEVEL SECURITY;
CREATE POLICY "service_role_all_access" ON [테이블명]
FOR ALL TO service_role USING (true) WITH CHECK (true);
```
