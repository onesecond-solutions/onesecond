| 인디케이터 | `↓` | 다음 섹션(기존 Hero) 유도 |

## 5. 디자인 스펙

### 5-1. 레이아웃

- **섹션 높이**: PC `min-height: 100vh` / 모바일 `min-height: 80vh` 허용
- **정렬**: 화면 정중앙 (가로·세로 모두 중앙)
- **배경**: 아이보리 단색 (--cream 또는 동등 토큰)
- **일러스트·이미지 0**: 텍스트만으로 압도

### 5-2. 타이포그래피

| 요소 | PC | 모바일 | 색상 | 굵기 | 자간 |
|------|-----|--------|------|------|------|
| 1줄 (2025년 어느 날) | 24px | 18px | 그레이 | 500 | +1 |
| 2줄 (사실 진술) | 32px | 24px | 본문 | 600 | 0 |
| 3줄 (따옴표 인용) | **56px** | **40px** | **--brown-dark** | **800** | -1 |
| 4줄 (마무리) | 20px | 16px | 본문 | 400 | 0 |

### 5-3. 색상 (브라운 면적 축소 원칙)

```css
/* 브라운은 따옴표 인용문 1곳만 */
.pre-hero-quote {
  color: var(--brown-dark); /* 또는 #8B4513 등 기존 토큰 */
}

/* 나머지는 모두 본문 컬러 또는 서브 그레이 */
.pre-hero-time {
  color: var(--text-secondary); /* 그레이 */
}

.pre-hero-fact, .pre-hero-closing {
  color: var(--text-primary);
}
```

### 5-4. 여백

- 1줄 → 2줄: **여백 80px** (PC) / **48px** (모바일)
- 2줄 → 3줄: **여백 60px** (PC) / **36px** (모바일)
- 3줄 → 4줄: **여백 60px** (PC) / **36px** (모바일)
- 4줄 → 인디케이터: **여백 120px** (PC) / **64px** (모바일)

## 6. 인터랙션 (선택사항)

### 6-1. 페이드인 시퀀스

```
0.0s ─ 빈 화면 (아이보리 배경만)
0.5s ─ "2025년 어느 날" 페이드인 (천천히 0.8s)
1.5s ─ "현장 영업 16년..." 슬라이드업 (0.6s)
2.3s ─ "와, 이런 게 없었다고?" 페이드인 + 살짝 확대 (1.0s)
3.5s ─ "— 그래서 직접 만들었습니다" 페이드인
4.5s ─ 스크롤 인디케이터 등장 (무한 반복 ↓)
```

### 6-2. prefers-reduced-motion 준수

```css
@media (prefers-reduced-motion: reduce) {
  /* 모든 애니메이션 0.01s로 단축 */
}
```
