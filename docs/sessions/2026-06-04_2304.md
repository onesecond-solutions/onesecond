1. 새 랜덤 secret 생성(64자 hex) → `.cron_secret_new.txt`(gitignore 등록). 마스킹 보고만.
2. Supabase Secrets `CRON_SECRET` 교체(팀장님 Dashboard).

검증 난항 → 근본 원인 규명:

- **3함수 재배포**(extract-knowledge / send-push / charge-subscriptions, 임시 PAT → CLI `--use-api --no-verify-jwt`) 후 **검증 통과**: 새 값 200 / 옛 값·헤더 없음 401.
