**수정:** 함수 정의를 Phase 7b (`ALTER users ADD COLUMN insurer_id` 직후)로 이동
**영구 학습:** SECURITY DEFINER 함수가 다른 테이블 컬럼 참조 시, 함수 정의 시점에 해당 컬럼이 존재해야 함 (PostgreSQL 14+ 메타 검증). 컬럼 신설 → 함수 정의 순서 강제 필수.
**적용:** D-pre.7~.8 패턴 + Phase 1 후속 트랜잭션 + 미래 모든 마이그레이션
