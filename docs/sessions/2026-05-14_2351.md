#### 3. 마스터 전략 §15 정체성 영구 박힘 (가장 큰 본진)

**팀장님 본인 명시 박힘:**
> "딸깍은 그냥 하는 소리가 아니야. 원세컨드 = 딸깍으로 만들 거야."
> "원세컨드 네이밍도 중요하지만 사람들이 쉽게 말할 수 있는 네이밍이 필요했는데 딸깍이 제일 좋은거 같애."
> "신규 사용자들이 딸깍만 보고 클릭해보면 자동으로 학습되게끔 반복 패턴 학습 및 중독시키는거야."

§15 박힌 본문:
- **본진** = 원세컨드 = one + second = 1초 안 박힘. 딸깍 = 한 번 클릭으로 끝남 본질
- **두 자리 네이밍** = 원세컨드(브랜드) + 딸깍(일상 호칭) — 당근/배민 같은 줄임 호칭 본진
- **학습·중독 본진** = 반복 패턴 + 자기 발견 학습 + 중독 회로 (1초 안 박힘 + 즉시 피드백 + 일관 패턴)
- **회귀 신호 7건** = 다단계 / 3 step+ / 4 step+ / 두 번 클릭 trade-off / 자산 패턴 깨짐 / 튜토리얼 의존 / 즉시 피드백 부재

본 §15 = 매 세션 첫 진입 강제 자리 박힘 (CLAUDE.md 절대 프로토콜 §0 정합).
